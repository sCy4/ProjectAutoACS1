User Manual
01

概觀

SPCM 是一套常駐於系統匣的操作輔助工具，將「快遞服務系統」中數項重複且費工的操作，整合為四組快捷鍵；執行時自動完成視窗切換、欄位填寫與按鈕點擊，減少人工步驟與操作失誤，並於過程中暫時鎖定滑鼠以避免誤觸。

02

開始前須知

  • 請先開啟並登入「快遞服務系統」。導出單號、提單圖片、追蹤查詢三項功能皆以其為操作對象；若未開啟，按下快捷鍵時會以提示框告知。
  • 功能執行期間，滑鼠會暫時鎖定、游標顯示為忙碌狀態，屬正常現象；如需強制中斷，請按 F8（詳見第 3 頁）。
  • 系統一次僅執行一項功能，須待前一項完成後方能執行下一項。
03

快捷鍵總覽

快捷鍵功能用途
Ctrl+Win+M 導出單號 導出指定配送員當日簽收單號之 Excel 檔
Ctrl+Win+P 提單圖片上傳 執行提單圖片上傳流程並送出
Ctrl+Win+C 追蹤查詢 將單號帶入追蹤查詢並執行查詢
Ctrl+Win+S 送出簡訊 於電訊簡訊平台自動完成送出之各步驟

Win 為鍵盤左下角、Ctrl 與 Alt 之間的 Windows 標誌鍵。

04

功能詳解

Ctrl+Win+M

導出單號

輸入一組配送員編號，導出其當日簽收單號 Excel 檔。

操作步驟
  1. 確認「快遞服務系統」已開啟。
  2. 按下快捷鍵，畫面出現輸入框。
  3. 輸入配送員編號，按「執行」或 Enter。
  4. 程式自動開啟簽收監控、帶入今日日期、查詢並導出 EXCEL。
  5. 檔案自動儲存至桌面，檔名為「編號.月日」。

注意若簽收監控視窗已開啟，須先關閉再執行；查詢可能需數秒，過程請勿操作滑鼠。

Ctrl+Win+P

提單圖片上傳

自動完成提單圖片的上傳流程並送出。

操作步驟
  1. 確認「快遞服務系統」已開啟，且「提單圖片管理」視窗未開啟。
  2. 按下快捷鍵，程式自動完成選單操作、點擊「上傳」並送出。

注意若「提單圖片管理」已開啟，須先關閉；過程將鎖定滑鼠，請待其完成。

Ctrl+Win+C

追蹤查詢

將單號帶入「追蹤查詢」並查詢。提供單號有兩種方式：

操作方式
  • 反白選取：先以滑鼠反白欲查詢之單號，再按快捷鍵。
  • 沿用剪貼簿：未反白時，若剪貼簿內容為純英數、20 字以內，將直接採用。

注意須先開啟「追蹤查詢」子視窗；若剪貼簿內容非純英數或超過字數，將提示並取消。

Ctrl+Win+S

送出簡訊

於瀏覽器之「電訊簡訊平台」自動完成送出之各步驟。

操作步驟
  1. 自行於瀏覽器開啟「電訊簡訊平台」並完成前置作業，停留於送出起始頁。
  2. 確認目前作用中視窗即為該瀏覽器分頁，按下快捷鍵。
  3. 程式依序完成下一步、確認傳送、確定與收單確定。

注意程式不會自動切換至簡訊平台；若非處於該瀏覽器視窗，將停止並提示。

05

操作須知

重要滑鼠鎖定與緊急解除

功能執行期間，程式會暫時鎖定實體滑鼠並將游標顯示為忙碌狀態，以避免誤觸而中斷流程。如遇異常或需中途停止，請按 F8 立即解除鎖定。

說明單一流程執行

同一時間僅執行一項功能。若已有流程進行中，再按其他快捷鍵將顯示提示「目前已有一個流程正在執行」，待其完成後即可繼續。

提醒提示框即操作指引

發生狀況時，程式會以文字提示框說明問題與處理方式（例如請先開啟特定視窗），並不會默默失敗，依提示操作即可。

06

疑難排解

狀況處理方式
提示「找不到『快遞服務系統』視窗」 先開啟並登入快遞服務系統後再執行；導出單號、提單圖片、追蹤查詢皆需其開啟。
按 Ctrl + Win + S 無反應，或提示未在簡訊平台 先切換至瀏覽器之「電訊簡訊平台」分頁並停留於起始頁，且確認該分頁為目前作用中視窗。
追蹤查詢提示剪貼簿內容非純英數 先反白單號再按快捷鍵；或確認剪貼簿為純英數單號（20 字以內，不含中文、空格或符號）。
滑鼠暫時無法點擊、游標持續忙碌 表示流程進行中，請稍候其完成；如需強制中斷，按 F8 解除鎖定。
系統匣圖示與預期不符 圖示將沿用程式 EXE 本身之圖示；若仍顯示舊圖示，多為 Windows 圖示快取未更新，重新開機即可。
